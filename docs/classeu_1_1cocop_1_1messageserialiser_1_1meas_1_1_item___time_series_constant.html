Detailed Description

A class to serialise time series with a constant sampling interval. Please note that this implementation does not support this type in an Item_DataRecord.

In this module, the code has been derived from TimeseriesML 1.0 - XML Encoding of the Timeseries Profile of Observations and Measurements (OGC 15-042r3; please see the file "ref_and_license_ogc_tsml.txt").
