Detailed Description

A temporal filter to be utilised in requests.

In this module, the code has been derived from OGC Filter Encoding 2.0 Encoding Standard - With Corrigendum (OGC 09-026r2; please see the file "ref_and_license_ogc_filter.txt") and OGC(r) Sensor Observation Service Interface Standard (OGC 12-006; please see the file "ref_and_license_ogc_sos.txt").
