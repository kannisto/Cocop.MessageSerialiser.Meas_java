◆ supportsDataQualityInDataRecord()

boolean eu.cocop.messageserialiser.meas.Item.supportsDataQualityInDataRecord ( )
protected

Whether the item type supports data quality in a data record. The default is true; override this function if the type does not support data quality in a data record. The property exists to enable the discovery of conflicts as early as possible. Otherwise, the error of associating data quality would appear only just before serialisation.

Returns
True if supported, otherwise false.
